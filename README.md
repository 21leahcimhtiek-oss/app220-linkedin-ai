# LinkedAI

> AI LinkedIn profile and outreach optimizer

## Features
- Profile scoring
- Headline generator
- Connection templates
- Content calendar
- Engagement analytics

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- OpenAI GPT-4o-mini
- Stripe Subscriptions
- Vercel deployment

## Quick Start

```bash
npm install
cp .env.example .env.local
# Fill in API keys
npm run dev
```
